## DocuRaft - A Documentation Website

DocuRaft is a documentation website where I showed a simple documentation style layout, markdown parsing, dynamic document searching, nested routing with Next.js 14.

Open [http://docuraft.vercel.app](http://docuraft.vercel.app) to see the result.

**Note:** *Project is under development.*

## How to run

**Note:** *You must have Node.js and Next.js installed on your computer.*

Open your command prompt and follow below steps (ensure you have git on your pc): 

```bash
git clone https://github.com/mkmasudrana806/docuraft.git

cd docuraft

npm install

npm run dev
```

Visit: [http://localhost:3000/](http://localhost:3000/)
